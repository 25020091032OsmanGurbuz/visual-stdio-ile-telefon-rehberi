# 📱 Visual Studio ile Telefon Rehberi Uygulaması
Bu proje, C# programlama dili ve Visual Studio ortamı kullanılarak geliştirilmiş temel bir telefon rehberi yönetim sistemidir. Kullanıcıların kişilerini dijital ortamda saklamasına, listelemesine ve yönetmesine olanak tanır.
---

## ✨ Özellikler
Kişi Ekleme: Yeni kişileri isim ve numara bilgileriyle kaydetme.

Listeleme: Kayıtlı tüm kişileri düzenli bir formatta görüntüleme.

Kullanıcı Dostu Arayüz: Visual Studio araçlarıyla oluşturulmuş kolay kullanım.
---

## 🚀 Kurulum ve Çalıştırma
Projeyi yerel bilgisayarınızda çalıştırmak için şu adımları izleyin:

Bu depoyu (repository) bilgisayarınıza indirin veya klonlayın.

Klasör içindeki telefon rehberi.sln dosyasını Visual Studio ile açın.

Üst menüden "Start" butonuna basarak projeyi derleyin ve çalıştırın.

## 🛠️ Kullanılan Teknolojiler
Dil: C#
